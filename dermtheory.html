01 — The brief

Why these visuals.

Dermtheory is clinical skincare in a category overflowing with claims. The stock detail page did the basics — a title, a few bullets, a gallery — but it left every real question unanswered: What's actually in it? Is it backed by anything? Will it work for my skin?

The brief was to build an infographic set that does the convincing the listing copy can't — a sequence of visuals that reads less like an advert and more like a confident, evidence-led explainer. Credible enough to trust, clean enough to scan in seconds.

02 — The idea

Proof, in a scrollable order.

I treated the set as a narrative with a deliberate sequence — hook, problem, science, routine, results, range. Each visual earns the next scroll by resolving one doubt before the shopper has to think it.

A restrained clinical palette, consistent grid and a single light direction across every mockup make the range read as one considered system — so the visuals feel like the brand, not a template dropped onto the page.
