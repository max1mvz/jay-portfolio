DROP YOUR PROJECT IMAGES HERE
=============================

The website automatically uses an image from this folder for each brand card.
Just save a file with the matching name below (JPG preferred, PNG also works).
If a file is missing, that card shows the elegant gradient placeholder instead.

Filenames the site looks for (exact, lowercase):

  ease.jpg          -> Ease Healthcare
  dermtheory.jpg    -> Dermtheory
  klutch1.jpg       -> Klutch1
  saponify.jpg      -> Saponify
  pranarom.jpg      -> Pranarom
  steam-go.jpg      -> Steam & Go
  moonjax.jpg       -> Moonjax
  beemo.jpg         -> Beemo

Tips
----
- Best results: landscape or square images, ~1200px wide or larger.
- The cards crop to fill, centered. A clean storefront hero or a strong
  product mockup works great.
- To grab the web versions from your Google Site: open the site, right-click
  the image you want, choose "Save image as...", and save it here with the
  matching name above.
- Your original export files (from Photoshop/Illustrator) will look sharper
  than the compressed web versions if you have them handy.
